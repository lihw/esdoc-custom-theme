    Installation

    Basic setup

    This is only a theme for esdoc. To use it you need to install esdoc and esdoc-publish-html-plugin

    yarn add esdoc esdoc-publish-html-plugin esdoc-custom-theme -D

    Advanced setup

    It is highly recommended to add the following plugins too. They will come in handy when configuring this theme.

    • esdoc-brand-plugin
    • esdoc-inject-style-plugin
    • esdoc-standard-plugin
    yarn add esdoc-brand-plugin esdoc-inject-style-plugin esdoc-standard-plugin -D
